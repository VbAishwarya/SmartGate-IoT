# Sample videos for scenario/detection tests

- **Format:** MP4 (e.g. H.264).
- **Content:** Short clips (5–15 s), e.g. vehicle approaching, vehicle leaving, empty scene.
- **Naming:** e.g. `scenario_quick_pass.mp4`, `scenario_slow_approach.mp4`.
- **Resolution:** Same as production (e.g. 640×480 or 1280×720).

Tests will skip if no files are present.
