# Image Processing & OCR (contrib)

- **Image Processing & OCR Module.ipynb** — Jupyter notebook for plate OCR (Tesseract). Colab-friendly (drive mount, file upload).
- **number_plates/** — Sample plate images for OCR; used by `tests/vision/` and `src/vision`.

Main app uses a mock sensor and does not run camera/OCR. For vision tests and dashboard plate check, see [SETUP.md](../../SETUP.md) and [docs/TESTING.md](../../docs/TESTING.md).
